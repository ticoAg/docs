## 行内代码

要把某个 `word` 或 `phrase` 标记为代码，请使用反引号（`）包裹。

```
To denote a `word` or `phrase` as code, enclose it in backticks (`).
```

## 代码块

使用 [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)：用三个反引号包裹代码，并在开头反引号后标注语言以获得语法高亮。你也可以在语言后追加文件名。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
