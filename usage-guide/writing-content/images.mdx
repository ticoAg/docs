## 图片

### 使用 Markdown

[Markdown 图片语法](https://www.markdownguide.org/basic-syntax/#images) 支持用如下方式插入图片：

```md
![title](/path/image.jpg)
```

注意：图片文件大小必须小于 5MB。否则建议使用 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 等服务托管，再通过 URL 进行嵌入。

### 使用 HTML 标签

如果你需要更高的自定义能力，也可以使用 HTML 标签插入图片：

```html
<img height="200" src="/path/image.jpg" />
```

## 嵌入与 HTML 元素

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/4KzFe50RQkQ"
  title="YouTube video player"
  frameBorder="0"
  allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
  allowFullScreen
  style={{ width: '100%', borderRadius: '0.5rem' }}
></iframe>

<br />

<Tip>

Mintlify 支持在 Markdown 中使用 [HTML 标签](https://www.markdownguide.org/basic-syntax/#html)。如果你更习惯 HTML，或需要更强的表现力，这会非常有用。

</Tip>

### iFrames

在文档中加载另一个 HTML 页面。最常见的用途是嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
